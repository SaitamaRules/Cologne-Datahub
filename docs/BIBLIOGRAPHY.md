# Bibliography & References

## Data sources

- **Cologne Geoportal** — Public tree registry (Baumkataster) served via WFS. <https://geoportal.stadt-koeln.de/>
- **OGC WFS 2.0** — Web Feature Service specification. <https://www.ogc.org/standards/wfs>
- **RFC 7946** — The GeoJSON Format. <https://datatracker.ietf.org/doc/html/rfc7946>

## Coordinate reference systems

- **EPSG:25832** — UTM Zone 32N, the projection used by the Cologne WFS. <https://epsg.io/25832>
- **EPSG:4326** — WGS84, used by the application and MongoDB's `2dsphere` index. <https://epsg.io/4326>
- **proj4js** — JavaScript library for reprojecting coordinates between systems. <https://github.com/proj4js/proj4js>

## Runtime and framework

- **Deno** — Official documentation. <https://docs.deno.com/>
- **Hono** — Official documentation. <https://hono.dev/>

## Databases

- **PostgreSQL 16** — Official documentation. <https://www.postgresql.org/docs/16/>
- **MongoDB 7** — Official documentation. <https://www.mongodb.com/docs/>
- **MongoDB geospatial queries** — `$geoNear` and `2dsphere` indexes. <https://www.mongodb.com/docs/manual/geospatial-queries/>
