# Security Policy

## Supported Versions

This project is in active development as part of an academic final project
(Trabajo de Fin de Ciclo — ASIR). Only the latest version on the `main`
branch is considered supported.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| others  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately rather
than opening a public issue:

- **Preferred:** open a draft security advisory at
  <https://github.com/SaitamaRules/Cologne-Datahub/security/advisories/new>.
- **Alternative:** contact the repository owner through their GitHub profile.

You can expect an initial response within 7 days. If the report is confirmed
as a valid vulnerability, a fix will be published on `main` and documented in
`CHANGELOG.md` under the `Security` section.
